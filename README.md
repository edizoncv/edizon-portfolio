# Edizon CV · Data Leader Portfolio

Professional data-driven portfolio built with React + Vite + Tailwind CSS + Framer Motion. This project features a high-fidelity dark theme inspired by premium modular designs, optimized for performance and storytelling through data.

## 🚀 Techo Stack
- **Framework**: React 18 (Vite)
- **Animations**: Framer Motion (Scroll-driven, Parallax, 3D Effects)
- **Styling**: Tailwind CSS (Custom Dark Theme)
- **Icons**: Lucide React
- **Deployment**: Cloudflare Pages

## 🛠 Features
- **Hero Section**: Sticky parallax effects with a strategic "Apple Style" storytelling phrase.
- **Services**: Dynamic 60/40 layout with sticky image reveal and clean typography.
- **Soluciones de Impacto**: Interactive project showcase featuring Power BI dashboards and predictive models.
- **Integrations**: Complex SVG animated diagram with real-time "electric" flow effects.
- **Testimonials**: Infinite marquee carousel with glassmorphism cards.
- **Contact**: Direct conversion via Calendly and WhatsApp integrations.

## 📦 Deployment to Cloudflare Pages
1. Connect this repository to your Cloudflare account.
2. Build Settings:
   - **Framework Preset**: Vite
   - **Build Command**: `npm run build`
   - **Build Output Directory**: `dist`
3. Environment Variables:
   - Ensure `NODE_VERSION` is set to `18` or higher if necessary.
